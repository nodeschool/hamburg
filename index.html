Learn Coding

Du wolltest schon immer mal programmieren? Oder willst endlich mal NodeJS ausprobieren? Nimm teil und lerne das Programmieren mit NodeJS ganz einfach - versprochen!Es hilt, wenn du schonmal etwas mit JavaScript gemacht hast - wenn nicht komm trotzdem! Wir helfen gern.

Join us and learn coding with us. You are going to learn the basics of Node.js. It's a plus if you know some JavaScript beforehand, but if not we can help you out.

Self-Guided

Du lernst mit kleinen Programmen, die du selbst schreibst - keine langweiligen Vorträge! Alle Aufgaben sind Open Source und online unter nodeschool.io verfügbar, so kannst du auch ganz einfach nach der NodeSchool selber weiter lernen.

You will go through a set of self-paced challenges. No boring talks. All challenges are Open Source and available online on nodeschool.io. This means you can continue learning after the event.

For Everyone

Wir finden programmieren großarting und möchten ein Event für alle machen. Das bedeutet, wir freuen uns über alle Teilnehmer - ganz gleich wie fitt ihr schon mit JavaScript oder NodeJs seid. Wir erwarten aber auch, dass alle genauso großartig zueinaner sind und unseren Code of Conduct respektieren.

We think coding is awesome and everyone should feel welcome at our event. This includes that we are expecting all participants to be awesome to each other and respect the Code of Conduct.

2.10.2017 from 15:00 to 20:00-ish at codecentric AG, Große Elbstr. 14

How to attend?

This is an Open Session! Just come by. We might need your name logged due to safety concerns, but thats it. At the location there is a limit. We won't announce the limit, but if we approximately reach the limit we are going to close the doors and communicate that via @NodeHamburg!

How to prepare?

Bring your hardware! Having Node.js and npm preinstalled is a plus. There is also Cloud9 IDE, which offers a free, fully integrated workspace.

Questions? Feedback? Want to help out

Just open an issue on GitHub and join the discussion. If you are not on GitHub yet, create a free account and check out the Git-it workshop. It will help you to get started. You can also chat with us at on Slack! Check in! Goto #nodeschoolhh
